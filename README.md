## Development

#### install requirements

```cmd
pip install -r requirements.txt
```

#### run the server

- For linux users
```cmd
python src/manage.py runserver
```

- For windows users
```cmd
python src\manage.py runserver
```

go to 127.0.0.1:8000 to view the site.
